## ขั้นตอน preprocessing

1. อ่าน `REBA_Source_Labels` จาก workbook และ export เป็น `data/research/expert_labels/reba2_training_ready_20260603.csv`.
2. อ่าน `ISO11228_Source_Labels` จาก workbook และ export เฉพาะแถว `data_quality_flag=OK` เป็น `data/research/expert_labels/iso11228_training_ready_20260603.csv`.
3. แถว ISO ที่ถูกกันออกก่อน train คือแถวที่คะแนนรวมเป็นศูนย์หรือ activity mapping ยังต้องให้ทีมวิจัยยืนยัน เช่น `REVIEW_ZERO_OR_MISSING_TOTAL` และ `REVIEW_ACTIVITY_MAPPING`.
4. Logistic Regression ใช้ MoveNet 51 features แล้วเพิ่ม engineered REBA angle features รวมเป็น 71 features.
5. XGBoost ONNX ใช้ MoveNet raw 51 features เพื่อให้ Flutter ONNX predictor ส่ง input ได้ตรง schema เดิม.
